Install Akamatis Firmware

Connect your device to the computer using a USB data cable, then follow the steps below.

  1. Connect the Akamatis device to your computer.
  2. Click Install Firmware.
  3. Select the serial port associated with the device.
  4. Keep the device connected until installation is complete.
This browser does not support firmware installation. Please use Google Chrome or Microsoft Edge on a desktop computer.
Important: Confirm that the selected device model matches the model shown on the product label.
